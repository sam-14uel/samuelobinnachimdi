AfriWallStreet

Project Overview

AfriWallStreet is a comprehensive social media platform designed specifically for the African market. The platform incorporates a wide range of features to provide users with a seamless and engaging social experience.

Key Features

  • Real-time Chat: Implemented using Django Channels and WebSockets to enable instant messaging between users.
  • Online/Offline Tracking: Real-time status updates showing when users are active on the platform.
  • AI Chatbot: Integrated intelligent assistant that helps users navigate the platform and provides automated support.
  • Advanced Search: Powerful search functionality allowing users to find content, users, and topics quickly.
  • Posting Features: Rich media posting capabilities, including text, images, videos, and links.
  • Notifications: Real-time notification system for interactions, messages, and platform updates.

Technologies Used

  • Backend: Django framework with Django REST Framework for API endpoints
  • Database: PostgreSQL for robust data storage and complex queries
  • Caching: Redis for improved performance and real-time features
  • WebSockets: Django Channels for real-time communication
  • Server: Daphne ASGI server for handling both HTTP and WebSocket protocols
  • Frontend: HTML, CSS, JavaScript with reactive components

Challenges & Solutions

One of the main challenges was implementing a scalable real-time messaging system. I solved this by using Django Channels with Redis as the channel layer, allowing for efficient message broadcasting and private messaging capabilities that could scale with user growth.

AI-powered Web Scraper API

Project Overview

This project creates a sophisticated web scraping system that collects data from various websites and provides an API for AI-powered queries against the collected information. It enables users to extract insights and analyze trends across multiple web sources.

Key Features

  • Intelligent Scraping: Advanced algorithms that can navigate complex websites and extract structured data.
  • Natural Language Queries: Users can ask questions in plain English and get relevant data from the scraped content.
  • RESTful API: Well-documented API endpoints for integrating with other applications.
  • Data Processing: Automated cleaning, normalization, and categorization of scraped data.
  • Scheduled Scraping: Configurable scheduling to keep data fresh and up-to-date.

Technologies Used

  • Web Scraping: Python with BeautifulSoup and Selenium
  • API Development: Django REST Framework
  • AI Components: Natural language processing for query understanding
  • Database: PostgreSQL for data storage
  • Task Scheduling: Celery with Redis for job queuing

Applications

This system has been used for market research, competitor analysis, and trend identification. It helps businesses make data-driven decisions by providing them with up-to-date information from across the web, all accessible through simple, intuitive queries.
